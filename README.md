## 📋 Project Description

The **Student Registration & Accommodation Approval System (SRAAS)** is a comprehensive web application designed for **Divine Word University (DWU)**. It streamlines student registration, finance verification, residency status management, and accommodation allocation.

The system supports both **initial registration** and **post-registration residency status changes** (Non-Residential → Residential), requiring manual payment verification by the Finance Office — reflecting real university processes.

---

## ✨ Key Features

### Student Features
- Secure user login
- View personal details and residency status
- Check account balance and fee status
- Request change from Non-Residential to Residential status

### Staff Features
- **Finance Office**: Verify payments and update residency status
- **Accommodation Office**: Manage room allocation
- **Admin**: System oversight and user management

### System Highlights
- Role-based access control (Student, Finance, Admin, Accommodation)
- Real-time status updates
- Support for residency status change workflow
- Responsive and user-friendly interface

---

## 🛠️ Technology Stack

- **Backend**: Node.js + Express.js
- **Frontend**: EJS (Embedded JavaScript) + HTML5 + CSS3
- **Session Management**: express-session
- **Database**: In-memory (easily upgradable to MySQL)
- **Development Tools**: VS Code, Git, GitHub
